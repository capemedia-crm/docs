## Overview

Cape Digital exposes its platform through a set of versioned REST APIs. All services share the same authentication model and base domain.

**Base domain:** `https://api.capedigital.co.ke`  
**API version:** `v1` — all endpoints are prefixed with `/api/v1/`

## Authentication

All endpoints require a Keycloak JWT Bearer token. See the [Authentication](/authentication) guide.

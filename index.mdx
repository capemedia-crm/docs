## Overview

The Commercial Service API provides endpoints for managing leads, campaigns, partners, and CTS (client tracking) workflows across the Cape Digital platform.

**Base URL**
```
https://api.capedigital.co.ke/commercial
```

**Current version:** `v1`  
All endpoints are prefixed with `/api/v1/`.

## Authentication

All API requests require a valid JWT token issued by Keycloak. See the [Authentication](/authentication) guide to obtain and use tokens.
